A simple devotional Android app to read **Hanuman Chalisa** with a clean, distraction-free interface.


## ✨ Features

* 📖 Full Chalisa text
* ⬅️ Next / Previous navigation
* 🎨 Clean typography for comfortable reading
* ⚡ Lightweight and fast
* 🧘 Minimal and distraction-free design
